# smltool
Quick 'n dirty hack to parse a smart-message-language data.

## Dependencies
You will need Python3 to run this script

## Setup
```
git clone https://github.com/xenobyte/smltool.git
cd smltool
```

Adjust IP_ADDRESS and PORT in the smltool.py

## Run
```
chmod +x smltool.py
python smltool.py
```
